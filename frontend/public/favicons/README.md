Place your favicon assets in this folder using the common naming conventions below.

Suggested filenames (drop these into this folder):
- apple-touch-icon.png                (180x180)
- favicon-32x32.png                   (32x32)
- favicon-16x16.png                   (16x16)
- android-chrome-192x192.png          (192x192)
- android-chrome-512x512.png          (512x512)
- safari-pinned-tab.svg               (SVG, with transparent fill)
- mstile-150x150.png                  (150x150 for Windows tile)
- site.webmanifest                    (already provided)
- browserconfig.xml                   (already provided)

After uploading, your favicons will be available at `/favicons/...` and the HTML
head already references these files. If you use different filenames, update
`frontend/index.html` accordingly.
